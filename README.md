# React To-Do List App

A simple and interactive To-Do List application built using React and Vite.

This project was created as part of a React assignment to practice:
- Functional Components
- State Management
- Props
- Event Handling

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit existing tasks

## Technologies Used

- React JS
- Vite
- JavaScript
- CSS

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/ramrohith999/to-do-list.git
```

### 2. Open the Project Folder

```bash
cd to-do-list
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

The application will run on:

```bash
http://localhost:5173
```

## Functionalities Implemented

### Add Task
Users can add a new task using the input field.

### Complete Task
Users can mark tasks as completed or undo them.

### Edit Task
Users can edit existing tasks.

### Delete Task
Users can remove tasks from the list.
